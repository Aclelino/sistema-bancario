# Sistema Bancário V1 #

Nesse projeto foi realizado para simular um sistema de banco
com as seguintes função.

* Deposito
* Saque
* Extrato

o usúario tem o limite de 3 saques por dia,
e fazer saque de até R$ 500.00 no dia.

## Requisitos

Python versão 3
